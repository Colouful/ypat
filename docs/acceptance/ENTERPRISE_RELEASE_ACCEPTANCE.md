## CLI 门禁

当前环境无法通过网络克隆私有仓库，以下命令尚未在当前最终 SHA 上独立执行：

```bash
cd frontend
rm -rf node_modules
npm ci
npm run type-check
npm run lint
npm run lint:all
npm run test
npm run build:h5
npm run build:mp-weixin

cd ../backend
mvn -pl system-wap,system-domain -am test -DskipTests
```
