## 技术决策记录

| 决策 | 选择 | 理由 |
|------|------|------|
| 前端框架 | UniApp 3.0 + Vue 3 + TS | 多端发布，Vue 3 生态成熟 |
| 构建工具 | Vite | 快速热更新，比 webpack 快 10x |
| 状态管理 | 待定 (Pinia 推荐) | Vue 3 官方推荐，比 Vuex 更轻量 |
| UI 风格 | Keep 风格 (绿色主调) | 设计稿确认，清爽简洁 |
| 样式方案 | SCSS + CSS 变量 | uni.scss 全局变量，组件 scoped |
| 请求封装 | 封装 uni.request | 统一拦截器、Token 注入、错误处理 |
