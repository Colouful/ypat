本轮完成代码级自动化测试、WAP 鉴权冒烟、构建产物扫描和 migration(数据库迁移) 验证；未完成真实预发环境完整业务联调。未完成项均需要正式预发依赖、测试账号、微信平台配置或支付沙箱/测试商户。

## 已验证的关键降级

- 反馈 Redis 异常策略通过后端测试覆盖：Redis 不可用时记录 warn(警告) 并 fail-open(失败时放行)，不打印联系方式。
- 生产构建地址扫描发现 HTTP IP，按预期阻断正式发布。
- WAP 私有接口匿名访问被拒绝，避免前端裸 userid 绕过鉴权。

## 需要预发补跑的真实链路

1. 微信登录：授权、首次完善资料、退出登录、Token 失效。
2. 发布约拍：图片上传、城市地点、拍拍豆、我的发布。
3. 报名与消息：A 发布、B 报名、A 收消息、B sent 跳约拍详情。
4. 联系方式：不足、正常解锁、重复解锁不重复扣费。
5. 意见反馈：预发库建表后正常提交和 60 秒限频。
6. 实名认证：两照提交、审核通过、审核拒绝、重新提交。
7. 钱包支付：商品列表、测试订单、取消、失败、账单和收支记录。
